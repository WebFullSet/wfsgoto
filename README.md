# Usage

Implementing wfsgoto.js into your project is easy.
<br>

Add script right before closing ``</body>`` tag:

```html
<script src="https://your_domain.com/js/wfsgoto.js"></script>
<script>
    const WebGoTo = new WFSGoTo({
        conteiner: '#header'
    })
</script>
```
